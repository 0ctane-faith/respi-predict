# respi-predict
Multimodal respiratory disease risk prediction, analysis, and forecasting system using chest X-rays, EHR records, patient questionnaire data, and machine learning models.
RespiPredict is an AI-assisted healthcare prototype that predicts and analyzes respiratory disease risk using multiple input sources including chest X-rays, EHR/Excel records, breathing audio concepts, and patient bio-data. The system includes ML model pipelines, synthetic EHR data generation, forecasting, multimodal fusion, and a Figma-based UI prototype.
